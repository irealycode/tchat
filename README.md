# Tchat
### python ```AES256``` encrypted sockets based chat

#### start server:
  ```
  $ python server.py -H [HOST] -P [PORT]
  ```
#### output:
  ```
  Listening on  [HOST] : [PORT] ...
  ```


#### connect to server:
  ```
  $ python client.py -H [HOST] -P [PORT] -S [PASSWORD]
  ```
#### output:
  ```
  username: [enter your username here]
  ```
